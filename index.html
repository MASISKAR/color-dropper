Welcome to the Color Dropper Tool!

How to Use:

  1. Upload Your Image: Click the "Choose file" button to select and upload your image. The image will appear on the canvas.
  2. Activate the Dropper: Click the "Color Dropper" button to enable the color dropper tool. Your cursor will change to a dropper icon.
  3. Pick a Color: Hover over any area of the image with the dropper tool activated. A zoom circle will appear, magnifying the area under the cursor and displaying the color in that region.
  4. View the Color: The color hex code will be shown inside the zoom circle. Click on the area to capture and display the color hex code in the header. The color hex code will be automatically copied to your clipboard.
  5. Disable the Dropper: Click the "Color Dropper" button again to disable the tool and return to the default cursor.
